# amazon-js-project
# 🛒 Amazon Clone – JavaScript Learning Project

## 👋 About This Project

This project is a **simple Amazon-style e-commerce website** built while I was learning **JavaScript**.

I followed a **22-hour JavaScript tutorial by SuperSimpleDev** where the instructor writes the code step-by-step and explains every concept. While watching the video, I **did not copy-paste the code**. Instead, I tried to **understand each concept and write the code myself on my computer**.
Whenever something was provided in the tutorial description (like assets or starter files), I used those resources, but the **JavaScript logic was written by me while learning and practicing along with the video**.
This project helped me understand how a real interactive website works using **HTML, CSS, and JavaScript**.

# ✨ Features

This project includes several pages and features similar to a basic e-commerce website:

* 🛍️ Product listing page
* ➕ Add items to cart
* 🧾 Checkout page
* 📦 Orders page
* 🚚 Order tracking page
* 🔄 Dynamic page updates using JavaScript

The UI is inspired by Amazon but simplified for learning purposes.

---

# 🛠️ Technologies Used

* **HTML** – page structure
* **CSS** – styling and layout
* **JavaScript (Vanilla JS)** – application logic

The project was built **without any frameworks**, which helped me focus on understanding **core JavaScript concepts**.
